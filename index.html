🐾 Meow Bot User Features

Meow Features:

  • 🐱 Meow & Bark Counting: Keep track of your playful nonsense with /meow and /barkcount
  • 🏆 Server Leaderboards: View server rankings with /meowboard
  • 🌐 Global Leaderboards: Track your meows across all servers with /globalmeowboard

Communication & Social:

  • 💌 Personal Reminders: Never forget important stuff with /remind
  • 🎭 Anonymous Confessions: Submit confessions anonymously with /confess
  • 📊 Polls: Create interactive polls with up to 10 options using /poll
  • 🧾 Quote System: Save and share memorable quotes (beta) with /randomquote
  • 🖼️ Avatar Display: Show anyone's Discord avatar in full size with /avatar

🛠️ Meow Bot Admin Features

(Features that require admin permissions)

Server Configuration:

  • 💌 Welcome Messages: Customizable greetings for new members—toggle on/off with /config_welcome
  • 🐱 Meow Rate Limiting: Control spam by capping meows per message with /config_meow_limits
  • 🌐 Global Leaderboard Participation: Opt in/out of cross-server leaderboards
  • 🎭 Anonymous Confessions: Toggle confession feature and set channels with /config_confess

Channel Management:

  • 💬 Quote Channel Setup: Configure where quotes are stored and retrieved
  • 🎮 Twitch Alerts: Set up notifications for favorite streamers (configurable per server)
  • 🎭 Reaction Roles: Assign roles via reactions (beta/work in progress)

Moderation Tools:

  • 🧹 Message Purging: Clear up to 100 messages at once with /clear
  • 🔧 Server Setup: Interactive configuration menu with /setup
