🐱 Multi-Server Meow Bot - What Can I Do?

🎯 The Basics

  • Works in unlimited Discord servers at once
  • Each server gets its own settings and data
  • Uses modern slash commands (type / to see them)
  • Easy setup with `/setup` command

😸 Fun & Games

  • Count "meows" in chat - see who's the most cat-like!
  • Server leaderboards to compete with friends
  • Optional global leaderboards across all servers
  • Anti-dog feature - get "hissed" at for saying "woof"
  • Spam protection to keep things fair

  • Customize messages with your own text

  • Integrated whitelist support - no separate commands required

  • Add reactions with /addreaction - includes whitelist in one command
  • Bot remembers your selected message per server
  • Works with any emoji (even custom server ones)
  • Easy management - list, clear, or fix broken setups

  • Get notified when streamers go live
  • Each server can track different streamers

  • Check who's live with `/twitch` command
  • Easy to add/remove streamers

⏰ Personal Reminders

  • Set reminders in plain English ("remind me in 2 hours")
  • Get reminded in DMs or specific channels
  • Reminders work even if bot restarts
  • See and cancel your reminders anytime

💬 Anonymous Confessions

  • Let members confess things anonymously
  • Each confession gets a number for organization
  • Optional private logs for moderators
  • Simple to set up confession channels

  • Create polls with up to 10 options
  • Add images to make polls more engaging
  • Automatic number reactions for voting
  • Removes invalid reactions to keep voting clean

🎨 Quote Sharing

  • Get random quotes from your quote collection
  • Choose where quotes get posted
  • Refresh button to get different random quotes
