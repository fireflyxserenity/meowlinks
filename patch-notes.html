🐱 Version 2.10.0 - "Meow Count Consistency Patch"

Release Date: December 14, 2025

▼

🔍 Issue Fixed: Server meow counts were inconsistent across different displays. Some servers showed different totals in chat vs. leaderboards due to counting departed members differently (~5-10% variance depending on member turnover).

🎯 Root Cause: Message responses counted ALL meows (including departed users), Global leaderboard only counted current members, This created a discrepancy in displayed totals

✅ Fixes Applied: Server Leaderboard (/meowboard): Now shows all-time total including departed users, Global Leaderboard (/globalmeowboard): Now shows accurate all-time server totals, Leaderboard rankings: Still only display current members in the top 10, Quotes: New cat/fish background added

🎉 Results: Message response meow count now matches /meowboard total, /meowboard total now matches /globalmeowboard entry, All historical meows are preserved and counted, No data loss - departed users' meows still matter!

✨ All meow counts are now consistent across all displays! *Thanks to user PoppaB for pointing this out*
