Version 1.0.0 - Initial Release

August 16, 2025

🌟 Core Features Introduced:

  • 📊 Smart XP System: Dual activity tracking for messages and voice chat with anti-spam protection
  • 🏆 Advanced Leaderboards: Time-filtered rankings (Today, Week, Month, Year, All Time) with interactive UI
  • 🎭 Automated Role Rewards: Level-based role assignment with 30 pre-designed cat-themed roles
  • 📈 Comprehensive Analytics: Daily activity tracking and progress visualization

🚀 Available Commands:

Member Commands:
  • /rank - Check current level, XP, and server ranking
  • /top - Interactive leaderboard with time period filters
  • /leaderboard - Classic top users display (1-25 users)
  • /activitylog - Detailed personal activity breakdown
  • /xp_guide - Complete XP system explanation
Admin Commands:
  • /activity_enable - Toggle the entire system on/off
  • /activity_config - Customize XP rates and settings
  • /set_level_role - Assign roles to specific levels
  • /remove_level_role - Remove role rewards
  • /level_roles - View all current role rewards
  • /setup_cat_roles - Auto-create 30 themed roles

⚙️ Customization Options:

  • 🎯 XP Per Message: Adjustable points for each message (default: 15)
  • 🎤 XP Per Voice Minute: Customizable voice chat rewards (default: 5)
  • ⏱️ Message Cooldown: Spam prevention with adjustable delays (default: 60s)
  • 📢 Level-Up Announcements: Choose announcement channel or disable
  • 🎭 Role Progression: Create unlimited custom level rewards

🎯 Technical Features:

  • 💾 SQLite Database: Reliable, fast, and efficient storage
  • ⚡ Optimized Performance: Built for servers with 200+ active users
  • 🔄 Real-Time Processing: Instant XP awards and level updates
  • 🛡️ Data Integrity: Secure database with parameterized queries
  • 🌍 Global Support: Works worldwide with UTC timestamps
