🔒 Privacy Policy

How we protect and handle your data across all Meow Bots services

Last Updated: March 24, 2026

This policy applies to all bots operated under the Meow Bots umbrella: Meow Bot, Cat Cafe Economy Bot, Meow Manager, and Twitch Meow Bot (collectively, "the Bots").

📝 Overview

Meow Bots ("we", "us") is committed to protecting your privacy. This Privacy Policy explains what data we collect, how we use it, and your rights regarding your information.

By using any of our Bots, you consent to the practices described in this policy. If you do not agree with our data practices, please do not use the Bots.

📊 Data We Collect

Automatically Collected Data (All Bots)

  • Discord User ID — To identify you in our systems
  • Discord Server ID — To track which server you're using the bot in
  • Usernames & Display Names — For leaderboards and profile features
  • Timestamps — When you interact with bot features

Interaction Data

  • Message Counts — Meow/bark counts per user per server (Meow Bot, Twitch Meow Bot)
  • Quotes & Confessions — Content created by users, including timestamps and user IDs (Meow Bot)
  • Economy Data — Coin balances, inventory items, profile settings, game stats, transaction history (Cat Cafe Economy)
  • Tickets & Threads — Ticket content, status, participants (Meow Manager)
  • Reminders — Reminder text, timestamps, recurrence settings (Meow Manager)
  • Leaderboard Data — Rankings and statistics across all bots

Server Configuration Data

  • Channel & Role IDs — Configured channels for features like confessions, logs, welcome messages, tickets
  • Server Settings — Preferences set by administrators including rate limits, features, and custom configurations
  • Twitch Streamer Info — Names and status of tracked streamers (Meow Bot)

🎯 How We Use Your Data

  • Core Functionality — Running the economy system, counting meows, managing tickets, sending reminders, and all bot features
  • Profiles & Leaderboards — Generating profiles and displaying rankings
  • Notifications — Twitch alerts, reminders, ticket updates, and scheduled messages
  • Server Administration — Allowing administrators to configure and manage bot features
  • Improvement — Fixing bugs, optimizing performance, and analyzing aggregate usage patterns

Anonymous Confessions

Confessions are displayed anonymously to other users. Server administrators and bot operators may access confession logs with user IDs attached for moderation purposes.

🔐 Data Storage & Security

  • Data is stored in secure databases with restricted access
  • Data is encrypted in transit (HTTPS/TLS)
  • Access is restricted to authorized personnel only
  • We do not store passwords — authentication relies on Discord's OAuth2 system
  • Regular security reviews and updates are performed

No security system is perfect. While we implement reasonable safeguards, we cannot guarantee absolute security. Users should not share sensitive personal information through the Bots.

🔄 Data Sharing

We do not sell, trade, or rent your personal information to third parties.

  • Server Administrators — Have access to confession logs, server statistics, ticket info, and configuration data as part of normal bot administration
  • Other Users — Can see public leaderboards, usernames, and rankings based on public activity
  • Discord — The Bots operate within Discord and are subject to Discord's data policies. Discord's Privacy Policy
  • Twitch — Limited streamer info is shared with Twitch's API for stream status verification
  • Legal — We may disclose data if required by law or to protect user safety

⏱️ Data Retention

📋 Summary of Data Practices

Data Type Retention
Server Configuration Until the bot is removed from the server
Economy Data (coins, items, profiles) As long as you use the bot, or until deletion requested
Meow/Bark Counts As long as you're in a server using the bot
Quotes & Confessions Until deleted by administrators or on request
Tickets & Threads Admin configured; closed tickets may be archived
Reminders Until completed or cancelled
Logs 30–90 days for debugging and improvement

If you request deletion, we will remove all your data upon verification. Leaderboard data may be retained for historical purposes (anonymized where possible).

✋ Your Rights

You have the right to:

  • Access — Request information about what personal data we hold about you
  • Deletion — Request complete removal of your data
  • Correction — Request updates to incorrect information
  • Data Portability — Request your data in a readable format
  • Opt-Out — Stop using bot features, leave servers, or ask administrators to remove your data

❓ Data Deletion Request Process

  1. Specify what data you want deleted (e.g., "all meow counts", "all economy data")
  2. Provide your Discord user ID and server ID (if server-specific)
  3. Contact your server administrator first
  4. If they can't help, contact us directly

Requests will be processed within 30 days where possible.

👧 Children's Privacy

The Bots are not intended for children under 13. We do not knowingly collect information from children under 13. Discord's Terms of Service require users to be 13+, and servers using the Bots should enforce age requirements.

🏛️ GDPR & CCPA

We process data based on legitimate interests (bot functionality), consent (by using the Bots), and legal obligations. EU users have additional rights including the right to erasure, restriction of processing, and the right to lodge a complaint with supervisory authorities. California residents have the right to know, delete, and opt-out under the CCPA. We do not sell personal information.

🔗 Third-Party Services

The Bots integrate with Discord and Twitch. We are not responsible for third-party privacy practices. The Bots do not use cookies or tracking pixels.

🔄 Changes to This Policy

We may update this Privacy Policy at any time. Significant changes will be announced when possible. Continued use of the Bots after changes constitutes acceptance of the updated policy.
