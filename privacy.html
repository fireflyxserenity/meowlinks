How we protect and handle your data

Last Updated: December 25, 2025

ğŸ“ Introduction

Meow Bot ("the Bot") is committed to protecting your privacy. This Privacy Policy explains what information we collect, how we use it, how we protect it, and your rights regarding your information.

ğŸ“Š Information We Collect

Discord User Information

  • User IDs: Unique Discord identifiers for users who interact with the Bot
  • Server IDs: Identifiers for servers where the Bot is installed
  • Usernames and Display Names: For leaderboard and public-facing features (when applicable)

Interaction Data

  • Message Counts: Number of "meow" and "bark" messages per user per server
  • Quotes: Text content of quotes created by users, including timestamps and user IDs
  • Confessions: Anonymous message text and associated metadata
  • Timestamps: When users interact with Bot features
  • Message Content: Limited processing of message content to identify triggers (meows, barks, etc.)

Server Configuration Data

  • Channel IDs: Configured channels for confessions, quotes, Twitch notifications, logs, etc.
  • Server Settings: Preferences set by server administrators including rate limits and features
  • Server Names: Display names for leaderboards and statistics
  • Welcome Messages: Custom welcome message text configured by administrators

Twitch Integration Data

  • Streamer Information: Names of Twitch streamers being tracked
  • Stream Status: Online/offline status and stream metadata
  • Stream Notifications: When and where stream notifications are sent

Leaderboard Data

  • Rankings: User rankings based on meow/bark counts
  • Statistics: Aggregated data about server activity
  • Participation Status: Whether servers opt into global leaderboards

ğŸ¯ How We Use Your Information

Primary Purposes

Your information is used to:

  • Provide Bot Functionality: Count meows/barks, store quotes, process confessions, send notifications
  • Generate Leaderboards: Create rankings and statistics for servers and globally
  • Send Notifications: Twitch stream alerts, reminders, and scheduled messages
  • Server Administration: Allow administrators to configure and manage Bot features
  • Analytics: Track feature usage to improve Bot performance (aggregate, non-identifying data)

Anonymous Confessions

Confessions are displayed anonymously to other users. Server administrators and Bot creators may access confession logs with user IDs attached for moderation purposes. User IDs are stored to prevent duplicate confessions and enable administrator review.

Quote Attribution

Quotes are stored with your user ID but can be accessed anonymously through commands. Your username is recorded as the creator but may not be visible depending on server configuration.

ğŸ” Data Storage and Security

Security Measures

We employ security measures including:

  • SQLite Database with Encryption Pragmas: Database uses Write-Ahead Logging (WAL) and integrity checks
  • Limited Access: Only authorized administrators can access raw database files
  • No Passwords Stored: User authentication relies on Discord's secure OAuth2 system

Limitations

We cannot guarantee absolute security. Despite our efforts, database files could be compromised if server security is breached. Users should not share sensitive personal information through the Bot.

ğŸ”„ Data Sharing

With Server Administrators

Server administrators have access to confession logs with user IDs, detailed server statistics, configuration information, and settings history. Administrators are responsible for handling this data appropriately.

With Other Users

In public leaderboards and statistics, we share usernames, message counts, rankings, and aggregated statistics. Users appearing on leaderboards do so based on their public activity in the server.

Third Parties

Twitch: If you use the Twitch integration, limited streamer information is shared with Twitch's API to verify stream status.

Discord: The Bot operates within Discord and is subject to Discord's data policies. Limited information may be shared with Discord as required by their platform (such as through their API).

No Other Sharing: We do not sell, rent, lease, or share your data with third parties for marketing or other purposes.

Legal Requirements

We may disclose information if required by law, necessary to prevent fraud, required to protect user safety, or to comply with Discord's Terms of Service.

â±ï¸ Data Retention

Retention Period

Data is retained for as long as you are a member of a server using the Bot, the Bot is installed in the server, or it is needed for the functionality of the Bot.

Deletion

You can request deletion of your data by contacting the server administrator of servers where you submitted content, requesting deletion of your meow/bark counts, or contacting the Bot creator for account-wide data deletion. Some data may be retained for backup purposes or may be recoverable from archives.
